## 0.1.0 (2026-03-13)

### Added
- Fluent guard clause API via `Guard.Against<T>()`
- Null, Default, OutOfRange, Negative, Zero checks
- String validations: NullOrEmpty, NullOrWhiteSpace, ShorterThan, LongerThan, InvalidEmail, InvalidUri
- Collection emptiness check
- Custom condition support via `Condition()`
